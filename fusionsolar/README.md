# FusionSolar MQTT Bridge - Home Assistant Add-on

Fetch real-time data from Huawei FusionSolar and publish to Home Assistant via MQTT.

## Quick Start

1. Configure your FusionSolar credentials
2. Set your MQTT broker details (usually `core-mosquitto`)
3. Find your Station DN from the FusionSolar web portal URL
4. Start the add-on
5. Sensors will be automatically discovered in Home Assistant

See DOCS.md for detailed instructions.
